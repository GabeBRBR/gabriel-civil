Dynamo: Renomeador de Vistas em Lote

Descrição da Ferramenta

Este script Dynamo para Revit visa agilizar o processo de renomear múltiplas vistas (plantas, cortes, elevações) de acordo com um padrão pré-definido, utilizando informações de parâmetros das próprias vistas.

Funcionalidades Principais:

  • Permite selecionar as vistas a serem renomeadas no projeto ativo.
  • Possibilidade de definir um prefixo e/ou sufixo padrão.
  • Utiliza valores de parâmetros existentes (como Nível, Nome da Disciplina, etc.) para compor o novo nome.
  • Interface simples no Dynamo Player para facilitar o uso por toda a equipe.

Como Usar:

  1. Abra o Revit e o projeto desejado.
  2. Inicie o Dynamo Player.
  3. Localize e execute o script `Renomeador_Vistas.dyn`.
  4. Na interface do Dynamo Player, selecione as vistas que deseja renomear.
  5. Configure os prefixos, sufixos e parâmetros a serem usados no novo nome (conforme as opções do script).
  6. Clique em "Executar" (ou "Run"). As vistas selecionadas serão renomeadas.
