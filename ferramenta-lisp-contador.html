LISP: Contador de Blocos Avançado

Descrição da Ferramenta

Esta rotina LISP para AutoCAD foi desenvolvida para automatizar a contagem de blocos específicos dentro de um desenho. É especialmente útil para quantitativos rápidos de elementos repetitivos como luminárias, pontos elétricos, peças sanitárias, etc.

Funcionalidades Principais:

  • Permite selecionar os nomes dos blocos a serem contados.
  • Opção de filtrar a contagem por layers específicos.
  • Gera um relatório simples na linha de comando ou (opcionalmente) em um arquivo de texto.
  • Ignora blocos em layers desligados ou congelados.

Como Usar:

  1. Carregue o arquivo `.LSP` no AutoCAD usando o comando `APPLOAD`.
  2. Digite o comando `CONTABLOCO` (ou o nome que você definir) na linha de comando.
  3. Siga as instruções: selecione os blocos ou digite seus nomes, escolha as layers (ou todas).
  4. Veja o resultado na linha de comando ou no arquivo gerado.
